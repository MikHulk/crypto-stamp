# CryptoStamp

CryptoStamp is a decentralized application that helps you port your writings 
to the Ethereum blockchain.

## Backend

The backend consists of a main CryptoStamp ERC-721 contract and on-the-fly deployed contracts 
linked to each token. 

It was developped with foundry. Instructions are given in its folder to test and deploy.

## Frontend

The frontend is a webapp developped with NextJS and RainbowKit. 

It's under development at this time. To participate to develomment follow instruction given in 
README under `crypto-stamp-frontend`.
